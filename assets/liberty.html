Airdrop System

The Airdrop System is a game mechanic where players interact with "airdrops" that pop up a reward roll GUI. Each airdrop provides a chance to win rewards, including high-value items, based on a probability system. I have used OOP to make each airdrop, that is being dropped its own instance. The airdrop spawn in a different place every time, so players need to go around the map to open each airdrop.
